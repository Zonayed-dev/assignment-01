# DevConf 2026 Landing Page

An exciting, responsive, and modern landing page built for **DevConf 2026**—a premier tech conference for developers, engineers, and tech enthusiasts. 

## 🚀 Live Deployment
You can view the live project here: [DevConf 2026 Live Page](https://zonayed-dev.github.io/assignment-01/)

---

## 🛠️ Features Included
* **Hero Banner Section:** A bold, full-screen interactive hero area welcoming attendees with custom typography and clean call-to-actions.
* **Speaker Lineup:** A beautifully structured 2-column responsive layout displaying prominent industry guest speakers, built using clean card styling.
* **Ticket Pricing Tier Matrix:** A flexible 3-tier pricing table showcasing Standard, Pro, and Team packages with custom feature lists.
* **AI Challenge Section (Hackathon Spotlight):** A premium, dark-mode focused feature section created with advanced layout structures highlighting an upcoming 24-hour developer hackathon.
* **Footer System:** A crisp, minimalist site footer housing basic copyright guidelines, structural terms, and privacy notes.

## 🤖 AI Challenge Documentation (10 Marks Section)

As required by the assignment guidelines, the **Hackathon Spotlight** section was ideated and styled with the assistance of AI to bring a creative, highly futuristic edge to the page that complements the tech conference theme.

### Exact Prompt Used:
> "Act as an expert frontend developer. I am building a website for an exciting tech conference known as DevConf2026. I am building a new section for a tech conference landing page called 'DevConf 2026'. The theme is modern, developer-centric, and futuristic. Please generate the HTML and semantic CSS for a 'Hackathon Spotlight' section. It should include a catchy title, a short description, three 'Tracks' (e.g., AI & Automation, Web3 & Decentralized, Open Source Innovation) styled as modern glassmorphism cards with icons, a prominent 'Register for Hackathon' call-to-action button, and a micro-countdown timer layout (just the static HTML/CSS structure for Days:Hours:Mins). Ensure the design is fully responsive, uses a dark mode aesthetic (like deep blues/grays with vibrant neon accents like cyan or purple), and fits seamlessly into a premium tech event page."

### How It Was Integrated:
1. **Semantic HTML Structure:** Integrated seamlessly directly into `index.html` alongside the existing conference sections.
2. **Responsive CSS Rules:** Appended cleanly into the main `style.css` stylesheet, featuring a custom grid layout system that translates fluidly across desktop, tablet, and mobile breakpoints.
3. **Thematic Harmony:** Tailored to sit nicely within a dark background context using cyan highlights to mimic high-end modern engineering platforms.
